2. Beschreibung der App

Die App CanisMentor bietet eine KI-gestützte Analyse von Hundeverhalten anhand von Videos sowie Trainingsvorschläge und Lerninhalte.

3. Registrierung und Nutzerkonto

Zur Nutzung bestimmter Funktionen kann ein Nutzerkonto erstellt werden, dabei werden folgende Daten verarbeitet:

  • E-Mail-Adresse
  • Passwort (verschlüsselt gespeichert)

Diese Daten ermöglichen den Zugang zum Nutzerkonto.

4. Hochgeladene Inhalte

Nutzer können Videos oder Texte zur Analyse hochladen. Diese Inhalte werden verarbeitet, um KI-gestützte Analysen und Trainingsvorschläge zu erstellen.

Die Inhalte können gespeichert werden, damit Nutzer später auf ihre Analysen zugreifen können.

5. Nutzung von KI-Diensten

Zur Analyse von Hundeverhalten nutzt die App KI-Dienste, insbesondere das Modell Gemini Flash von Google.

Dabei können folgende Daten verarbeitet werden:

  • hochgeladene Videos oder Bilder
  • Texteingaben des Nutzers
  • Analysebezogene Metadaten

Die Verarbeitung erfolgt ausschließlich zur Erstellung von Analyseergebnissen und Trainingsvorschlägen innerhalb der App.

Die übermittelten Inhalte werden nicht zum Training von KI-Modellen verwendet.

Der KI-Dienst wird über Server von Google bereitgestellt. Dabei kann eine Verarbeitung von Daten außerhalb der Europäischen Union erfolgen. Google verpflichtet sich zur Einhaltung der geltenden Datenschutzvorschriften.

6. Speicherung von Daten

Wir speichern bestimmte Daten, um Ihnen die Nutzung unserer App zu ermöglichen und die angebotenen Funktionen zu verbessern. Dazu gehören insbesondere:

  • Nutzerkontodaten, die Sie bei der Registrierung oder Nutzung Ihres Kontos angeben.
  • Analyseergebnisse, die durch die Verarbeitung Ihrer Aktivitäten und Inhalte entstehen.
  • Verhaltensmuster aus den Analysen.
  • Chatverläufe sowie Erkenntnisse aus Chatverläufen, um individuelle Empfehlungen und Inhalte zu verbessern.
  • Erstellte Pläne, Übungen und Übungsfeedback sowie Trainingsfortschritt, um Ihren Lern- und Trainingsverlauf zu dokumentieren und weiterzuentwickeln.

Wir verwenden diese Daten ausschließlich zur Verbesserung der App-Funktionen, zur Anpassung von Trainingsplänen und zur Optimierung Ihrer Nutzererfahrung.

7. Nutzung von Firebase-Diensten

Die App nutzt Dienste der Plattform Firebase der Google Ireland Limited, Gordon House, Barrow Street, Dublin 4, Irland. Firebase ist eine Entwicklungsplattform, die verschiedene Cloud-Dienste für mobile Anwendungen bereitstellt.

Firebase Authentication

Zur Registrierung und Anmeldung von Nutzern wird Firebase Authentication verwendet. Dabei werden folgende Daten verarbeitet:

  • E-Mail-Adresse
  • verschlüsseltes Passwort
  • technische Identifikationsdaten des Nutzerkontos

Die Verarbeitung erfolgt ausschließlich zum Zweck der sicheren Anmeldung und Verwaltung des Nutzerkontos.

Firebase Storage

Zur Speicherung von durch Nutzer hochgeladenen Inhalten (z. B. Videos oder Bilder zur Analyse) nutzt die App Firebase Storage.

Dabei können folgende Daten gespeichert werden:

  • hochgeladene Videos oder Bilder
  • Analyseergebnisse
  • technische Metadaten

Die Speicherung erfolgt, damit Nutzer auf ihre Inhalte und Analysen innerhalb der App zugreifen können.

2. Description of the App

The CanisMentor app provides AI-based analysis of dog behavior from videos as well as training suggestions and learning content.

3. Registration and User Account

Certain features require a user account, for which the following data is processed:

  • Email address
  • Password (stored encrypted)

These data enable access to the user account.

4. Uploaded Content

Users can upload videos or text for analysis. These contents are processed to generate AI-based analyses and training suggestions.

The content may be stored so that users can access their analyses later.

5. Use of AI Services

To analyze dog behavior, the app uses artificial intelligence services, in particular the Gemini Flash model provided by Google.

The following data may be processed:

  • uploaded videos or images
  • user text input
  • analysis-related metadata

The processing is carried out solely to generate analysis results and training recommendations within the app.

User content submitted for analysis is not used to train AI models.

The AI service is provided via Google servers. This may involve processing data outside the European Union. Google commits to complying with applicable data protection regulations.

6. Data Storage

We store certain data to enable the use of our app and to improve the services offered. This includes, in particular:

  • User account data provided during registration or account usage.
  • Analysis results generated from processing your activities and content.
  • Behavior patterns from analyses.
  • Chat histories and insights from chat histories to enhance personalized recommendations and content.
  • Created plans, exercises and exercise feedback, and training progress to document and develop your learning and training experience.

These data are used solely to improve app functionality, customize training plans, and optimize your user experience.

7. Use of Firebase Services

The app uses services of the platform Firebase, provided by Google Ireland Limited, Gordon House, Barrow Street, Dublin 4, Ireland. Firebase is a development platform that provides various cloud services for mobile applications.

Firebase Authentication

Firebase Authentication is used for user registration and login. The following data may be processed:

  • email address
  • encrypted password
  • technical identifiers related to the user account

This data is processed solely for secure authentication and management of the user account.

Firebase Storage

User-uploaded content (for example videos or images for analysis) may be stored using Firebase Storage.

The following data may be stored:

  • uploaded videos or images
  • analysis results
  • technical metadata

Storage is necessary so that users can access their content and analyses within the app.

Further information about Google's privacy practices: https://policies.google.com/privacy
