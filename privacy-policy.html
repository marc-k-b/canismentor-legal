2. Beschreibung der App

CanisMentor ist eine KI-gestützte Hundetrainings-App für iOS und Android. Sie bietet unter anderem folgende Funktionen:

  • KI-Analyse von Hundeverhalten anhand von Videos oder Texteingaben
  • Erstellung individueller, KI-generierter Trainingspläne
  • KI-gestützter Chat-Assistent für Trainingsfragen
  • Übungsbibliothek mit Fortschrittsverfolgung
  • KI-generierte Übungsbilder (Collagen)
  • KI-Sprachausgabe (Text-to-Speech) für Trainingshinweise
  • Push-Benachrichtigungen als Trainingserinnerungen
  • Kostenpflichtige Premium-Funktionen (Abonnement / Einmalkauf)

E-Mail und Passwort

Apple Sign-In (Sign in with Apple)

Nutzer können sich über ihr Apple-Konto anmelden. Dabei wird ein anonymisierter Nutzeridentifikator von Apple übermittelt. Eine E-Mail-Adresse kann wahlweise weitergegeben oder von Apple verschleiert werden. Die Datenverarbeitung durch Apple unterliegt der Datenschutzrichtlinie von Apple (apple.com/legal/privacy).

Google Sign-In

Nutzer können sich über ihr Google-Konto anmelden. Dabei können Name, E-Mail-Adresse und ein Profilbild von Google übermittelt werden. Die Datenverarbeitung durch Google unterliegt der Google-Datenschutzrichtlinie (policies.google.com/privacy).

  • Video- und Textfeedback zu Trainingseinheiten

8. Firebase-Dienste im Überblick

Die App nutzt mehrere Dienste der Plattform Firebase der Google Ireland Limited, Gordon House, Barrow Street, Dublin 4, Irland.

Firebase Authentication

Für die Registrierung und Anmeldung (E-Mail/Passwort, Apple Sign-In, Google Sign-In). Verarbeitet werden: E-Mail-Adresse, verschlüsseltes Passwort, technische Nutzerkennung (UID).

Firebase Firestore

Hauptdatenbank der App. Gespeichert werden: Nutzerprofil, Hundedaten, Trainingspläne, Analyseergebnisse, Chat-Verlauf, Push-Benachrichtigungseinstellungen, FCM-Token, Zeitstempel der letzten Aktivität (lastActiveAt) sowie Cooldown-Einstellungen für KI-Funktionen.

Firebase Storage

Für die Speicherung hochgeladener Dateien (Videos zur Analyse, Profilfotos).

Firebase Cloud Functions

Serverseitige Funktionen, die KI-Anfragen (Analysen, Trainingspläne, Chat, Bildgenerierung, Sprachausgabe) abwickeln und die App-Logik ausführen.

Firebase Cloud Messaging (FCM)

Für das Senden von Push-Benachrichtigungen (Trainingserinnerungen). Dabei werden Gerätetoken (FCM-Token) gespeichert und einem Nutzerprofil zugeordnet. Der Zeitstempel der letzten App-Nutzung (lastActiveAt) wird in Firestore gespeichert, um Inaktivitätserinnerungen zu steuern.

Push-Benachrichtigungen sind optional. Die Berechtigung kann im Betriebssystem der Geräte jederzeit widerrufen werden.

Weitere Informationen: policies.google.com/privacy

2. Description of the App

CanisMentor is an AI-powered dog training app for iOS and Android. Key features include:

  • AI-based analysis of dog behavior from videos or text descriptions
  • AI-generated personalized training plans
  • AI chat assistant for training questions
  • Exercise library with progress tracking
  • AI-generated exercise images (collages)
  • AI text-to-speech for training guidance
  • Push notification training reminders
  • Premium features via subscription or one-time purchase

A user account is required to use the app. Registration and sign-in can be performed in the following ways:

Email and Password

  • Email address
  • Password (stored encrypted)

Sign in with Apple

Users can sign in using their Apple ID. Apple provides an anonymized user identifier and, optionally, an email address (which may be relayed or hidden by Apple). Data processing by Apple is governed by Apple's Privacy Policy (apple.com/legal/privacy).

Google Sign-In

Users can sign in using their Google account. Google may provide a display name, email address, and profile picture. Data processing by Google is governed by Google's Privacy Policy (policies.google.com/privacy).

During onboarding and profile management, the following data is collected and stored in Firebase Firestore:

User Data

Dog Data

  • Dog's name
  • Breed
  • Age / birthday
  • Gender

7. Use of AI Services (Google Gemini)

AI-powered features are executed via Firebase Cloud Functions, which in turn use AI models provided by Google, in particular Google Gemini.

The following data may be transmitted to the AI service:

  • Videos or images for behavior and training analysis
  • User text input (problem descriptions, chat messages)
  • Dog profile data for context (name, breed, age, training goals)
  • Previous analysis results and training history (for follow-up analyses)

Submitted content is used exclusively to generate analysis results, training plans, exercise suggestions, images, and speech output within the app. User data is not used to train AI models.

Processing occurs on Google servers, which may be located outside the European Union. Google commits to complying with applicable data protection regulations. Further information: policies.google.com/privacy

8. Firebase Services Overview

The app uses several services from the Firebase platform of Google Ireland Limited, Gordon House, Barrow Street, Dublin 4, Ireland.

Firebase Authentication

Used for registration and sign-in (email/password, Apple Sign-In, Google Sign-In). Data processed: email address, encrypted password, technical user identifier (UID).

Firebase Firestore

The app's primary database. Stored data includes: user profile, dog data, training plans, analysis results, chat history, push notification settings, FCM tokens, last activity timestamp (lastActiveAt), and AI-function cooldown settings.

Firebase Storage

Used for storing uploaded files (videos for analysis, profile photos).

Firebase Cloud Functions

Server-side functions that handle AI requests (analyses, training plans, chat, image generation, speech generation) and execute app logic.

Firebase Cloud Messaging (FCM)

Used to send push notifications (training reminders). Device tokens (FCM tokens) are stored and associated with a user profile. The timestamp of the most recent app use (lastActiveAt) is stored in Firestore to manage inactivity reminders.

Push notifications are optional. The permission can be revoked at any time in the device's operating system settings.

Further information: policies.google.com/privacy
